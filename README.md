BioProject es una herramienta profesional de bioinformГЎtica diseГұada para el escaneo, identificaciГіn y anГЎlisis funcional de secuencias de ADN. Combina la potencia del motor local **NCBI BLAST+** con la conectividad global de la **API de NCBI (QBlast)**.

## Funcionalidades Principales

### 1. Escaneo Local
Permite realizar bГәsquedas ultra-rГЎpidas contra una base de datos propia almacenada en la carpeta `database/`. Ideal para identificar secuencias previamente conocidas o variantes locales sin necesidad de internet. Utiliza Гӯndices optimizados (`.nhr`, `.nin`, `.nsq`).

### 2. Escaneo Global (NCBI Worldwide)
Conecta directamente con los servidores de NCBI en Maryland, USA, para comparar una secuencia contra la base de datos **nt** (nucleГіtidos), la mГЎs grande del mundo. Identifica organismos, patГіgenos y genes con precisiГіn cientГӯfica (E-value).

### 3. Laboratorio de AnГЎlisis Funcional
MГЎs allГЎ de la bГәsqueda, este mГіdulo interpreta la secuencia:
- **TraducciГіn a ProteГӯna**: Simula el dogma central de la biologГӯa convirtiendo ADN en una cadena de AminoГЎcidos hasta encontrar un codГіn de parada.
- **Contenido GC**: Calcula la estabilidad termodinГЎmica de la secuencia, Гәtil para identificar firmas genГіmicas.

### 4. Historial Inteligente (JSON)
Todas las bГәsquedas y hallazgos se registran automГЎticamente en un archivo `history.json`, permitiendo auditorГӯa cientГӯfica y persistencia de datos para futuros anГЎlisis.

## Estructura del Proyecto

- `main.py`: Punto de entrada con menГә interactivo.
- `bio_project/core/`: Motores de bГәsqueda y lГіgica de anГЎlisis.
- `bio_project/cli/`: Interfaces de lГӯnea de comandos.
- `bio_project/database/`: AlmacГ©n de archivos FASTA.
- `bio_project/logs/`: Historial de consultas en formato JSON.

## Futuras Mejoras (Roadmap)

Este proyecto es una base sГіlida que puede expandirse en las siguientes direcciones:

1.  **IntegraciГіn de Scraping AutomГЎtico**: Crear un mГіdulo que extraiga automГЎticamente las Гәltimas secuencias de variantes de interГ©s (E.g., nuevas cepas de virus) desde repositorios pГәblicos como GISAID o NCBI Virus.
2.  **VisualizaciГіn FilogenГ©tica**: Implementar la generaciГіn de ГЎrboles filogenГ©ticos (`Phylo` de Biopython) para visualizar la distancia evolutiva entre los resultados encontrados.
3.  **DetecciГіn de Mutaciones EspecГӯficas**: AГұadir un comparador que resalte cambios puntuales (SNPs) entre la secuencia query y el "Top Hit" de la base de datos.
4.  **PredicciГіn de Estructura 3D**: Integrar enlaces o scripts para conectar las secuencias de proteГӯnas traducidas con herramientas de predicciГіn de plegamiento (como mini-implementaciones inspiradas en AlphaFold).
5.  **Interfaz Web (Dashboard)**: Migrar la CLI actual a un dashboard interactivo utilizando **Streamlit** o **Flask** para visualizar resultados con grГЎficos de barras y mapas de calor genГіmicos.
6.  **Soporte Multihilo**: Optimizar las bГәsquedas locales masivas utilizando procesamiento en paralelo para grandes archivos FASTA.
